# Security Policy

## Responsible Use
This tool is designed strictly for legitimate scam reporting.

❌ Do not use for harassment  
❌ Do not submit false reports  
❌ Do not automate submissions  
❌ Do not mass-report numbers  

Misuse may violate local laws or WhatsApp policies.

## Reporting a Vulnerability
If you discover a security issue:
- Open a GitHub issue with minimal details, OR
- Contact the maintainer privately if applicable

Do not publicly disclose vulnerabilities without coordination.
